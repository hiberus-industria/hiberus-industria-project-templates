===========================================
PACKAGE ICON PLACEHOLDER
===========================================

This file is a placeholder for the NuGet package icon.

REQUIRED ACTION:
- Create a PNG icon file named: icon.png
- Place it in the root directory (same location as this file)
- Delete this placeholder file

ICON REQUIREMENTS:
- Format: PNG
- Dimensions: 512x512 pixels (recommended)
- Minimum: 128x128 pixels
- Transparent background preferred for better integration
- Should include Hiberus branding/logo
- Keep file size reasonable (<200KB recommended)

DESIGN GUIDELINES:
- Use official Hiberus brand colors and logo
- Ensure icon is recognizable at small sizes (64x64)
- Avoid text that becomes illegible when scaled down
- Consider how it appears on both light and dark backgrounds

TECHNICAL NOTES:
- The .nuspec file references "icon.png"
- NuGet will display this icon in package listings
- Icon will be embedded in the .nupkg during pack operation

For brand guidelines and official logos, contact:
- Hiberus brand/marketing team
- Design department

Once icon.png is created, you can verify it's included:
  dotnet pack -c Release
  unzip -l artifacts/*.nupkg | grep icon.png
